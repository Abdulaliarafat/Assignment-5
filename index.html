Question-1: What are the different ways to select an element in the DOM?

Use getElementById, getElementsByClassName, getElementsByTagName, getElementsByName, querySelector, and querySelectorAll to select elements in DOM.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets/sets HTML, innerText gets/sets visible text, and textContent gets/sets all text including hidden.

Question-3: What is event delegation in the DOM?

Event delegation allows a parent element to handle events from its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling means an event starts from the target element and propagates up to its parents.

How do you create, add, and remove elements using JavaScript?

Use document.createElement(), appendChild(), removeChild(), or remove() to create, add, and remove elements.
